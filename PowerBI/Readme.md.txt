# Retail Sales Dashboard

## Project Overview

This project is an interactive Power BI dashboard designed to analyse retail sales performance across different products and regions.

## Objectives

* Track total sales performance
* Analyse product sales
* Compare regional sales performance
* Identify sales trends over time
* Provide interactive filtering for users

## Dashboard Features

* Total Sales KPI
* Total Quantity KPI
* Average Sales KPI
* Daily Sales Trend
* Regional Sales Analysis
* Total Products
* Product Filter
* Region Filter
* Date Filter

## Tools Used

* Microsoft Power BI
* DAX
* Excel

## Skills Demonstrated

* Data Visualisation
* Business Intelligence Reporting
* Dashboard Design
* Data Analysis
* KPI Development
* Interactive Reporting
